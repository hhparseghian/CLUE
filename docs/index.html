CLUE

Cross-platform Lightweight UI Engine — a C99 GUI toolkit with OpenGL ES 2 rendering.

C99 Wayland X11 DRM/KMS MIT License

Overview

CLUE is a retained-mode widget toolkit written in C99. It renders with OpenGL ES 2 via EGL and supports Wayland, X11, and DRM/KMS display backends. Include <clue/clue.h> to access the full API.

Quick Start

#include <clue/clue.h>

// Callback for button click
static void on_clicked(void *widget, void *data)
{
    clue_label_set_text((ClueLabel *)data, "Button clicked!");
}

int main(void)
{
    ClueApp *app = clue_app_new("My App", 400, 300);

    ClueBox *vbox = clue_box_new(CLUE_VERTICAL, 12);
    ClueLabel *label = clue_label_new("Hello, CLUE!");
    ClueButton *btn = clue_button_new("Click Me");
    clue_signal_connect(btn, "clicked", on_clicked, label);

    clue_container_add((ClueWidget *)vbox, (ClueWidget *)label);
    clue_container_add((ClueWidget *)vbox, (ClueWidget *)btn);

    clue_app_set_root(app, (ClueWidget *)vbox);
    clue_app_run(app);
    clue_app_destroy(app);
    return 0;
}

Alignment

ClueAlign
CLUE_ALIGN_START | CLUE_ALIGN_CENTER | CLUE_ALIGN_END

Set widget->base.style.h_align and .v_align on a box to control how children are positioned within the available space.

Expand

Set widget->base.style.hexpand = true to make a widget expand horizontally to fill its parent. Set .vexpand = true for vertical. When used inside a box, expanding children share extra space equally.

Other fields

Style fields can also be set directly: widget->base.style.bg_color, .fg_color, .corner_radius, .font, .margin_top, .margin_right, .margin_bottom, .margin_left.
